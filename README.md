# Projeto de Natal Interativo

Este é um pequeno projeto de front-end que criei para celebrar o Natal. A página apresenta uma cena de Natal com uma árvore, um elfo e presentes, com elementos interativos controlados por JavaScript.

## Funcionalidades

- **Árvore de Natal Iluminada:** Uma árvore de Natal estilizada com CSS, com luzes que podem ser acesas.
- **Animação de Luzes:** Ao clicar no botão, as luzes da árvore acendem em sequência e depois começam a piscar.
- **Mensagem de Natal:** Quando as luzes são acesas, uma mensagem de Natal aparece com um efeito de digitação.
- **Elfo Animado:** Um pequeno elfo com animações sutis, como piscar de olhos.
- **Links Sociais:** Os presentes sob a árvore contêm links para meus perfis no LinkedIn e GitHub.

## Como Executar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` em seu navegador de preferência.

## Tecnologias Utilizadas

- **HTML5:** Para a estrutura da página.
- **CSS3 (Sass):** Para a estilização e animações. O código foi compilado de arquivos `.scss`.
- **JavaScript:** Para a interatividade, como o controle das luzes e o efeito de digitação.
